TI-82 and TI-83 Calculator Games

To install these games, you need a TI Graphing Calculator (TI-82 or TI-83), GraphLink software, and a GraphLink.  Ti-82 games can be played on a TI-83, but not vica versa.

TI-83 Basic Games

  1. Boxs - I don't know what this is
  2. Tetris83 - Tetris
  3. CrapsII - a Craps game that I wrote
  4. Hangman - you type a word, your friend tries to guess it
  5. Haysguess - similar, pointless game
  6. LinkTalk - talk to somebody over the cord thing
  7. Memory - like Simon, you remember the order in which the numbers flash
  8. Password - you have to enter a 5-letter password whenever you turn on your calculator
  9. Word83 - a word processor for the calculator, saves documents as pics

 

TI-83 Assembly Games

In order to use these games, you must have one or all of the following:

  • Zasmload
  • ZSOS
  • ZGFXL
  • ZLib

The games:

  1. Breakout - a really cool breakout game
  2. Duckhunt - an assembly version
  3. Frogger - like the video game
  4. Jezzball - like the Win95 game where you try to trap a ball
  5. Mario - i've never played this
  6. Nibbles - the game where you are a snake and you eat crumbs
  7. Penguins - a cool game, but you have to have a lot of free memory
  8. Tunnel - a race game
